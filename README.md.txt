Tools Used
Selenium WebDriver: Browser automation
WebDriver Manager: Automatic driver setup
Python Logging: Test execution logging
Chrome Options: Headless mode/anti-detection configs
=========================================================
Steps to Run Automation
Install dependencies:
bash Copy
pip install selenium webdriver-manager
Download ChromeDriver (or let WebDriver Manager handle it)
Run the script:
bash Copy
python todoist_automation.py
Enter Todoist credentials when prompted
===========================================================
Assumptions
Todoist UI structure remains stable (selectors may need updates)
Test account has no rate limits
Chrome browser is installed
Internet connection is stable
